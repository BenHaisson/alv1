# AGENTS.md

Use Motion for every interaction.

No GSAP.

No CSS animations.

Transitions must feel like Porsche, Apple and Awwwards.

Animations should be:

- cinematic
- elegant
- smooth
- premium
- never bouncy
- never playful

Cars move with inertia.

Cards expand using layout animations.

Page transitions use shared `layoutId`.

Booking section should animate from bottom.

Fleet images should animate using `clip-path`.

Gallery uses `staggerChildren`.

Hover effects use spring animations.

Always use Motion.
